Our Model

The model was based using a Resnet50 model, training it with 8000 images, then tested using 200 images. Image data consisted of a variety of images of cats and dogs, evenly split between both species. A Resnet50 model was used as the our model built from scratch, using tensor-flow and keras, had a 75% accuracy, so to improve the accuarcy of the model, a pre-trained model was utlised.

Sources:

  • Kaggle Image Data - https://www.kaggle.com/chetankv/dogs-cats-images

Modules and Languages used:

  • python
  • tensor-flow
  • keras
  • Resnet50
